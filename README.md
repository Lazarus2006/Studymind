# StudyMind 🧠

StudyMind is an all-in-one, AI-powered desktop companion designed to streamline your learning and research workflows. Built entirely in Python using PyQt5, it seamlessly brings together document reading, structured note-taking, and intelligent AI assistance into a single, cohesive interface.

[![License: GPL v2](https://img.shields.io/badge/License-GPL%20v2-blue.svg)](https://gnu.org/licenses/gpl-2.0.html)

---

## ✨ Features

*   **🤖 Advanced AI Integration:** Fully powered by the modern `google.genai` SDK for lightning-fast, context-aware assistance, brainstorming, and explanations.
*   **📄 Integrated PDF Reader:** Read your textbooks, research papers, or lecture notes directly inside the application without switching tabs.
*   **✍️ Dynamic Notes Workspace:** A built-in text environment to quickly jot down summaries, key insights, and study guides side-by-side with your documents and chat.
*   **💻 Clean & Responsive UI:** Crafted with `PyQt5` and `QtWebEngineWidgets` to deliver a smooth, high-performance desktop experience.
*   **🔒 Local & Secure:** Your API keys and application configurations are saved securely on your local machine.

## 🛠️ Tech Stack

*   **Language:** Python 3.x
*   **GUI Framework:** PyQt5 (including QtWebEngineWidgets)
*   **AI Engine:** Google GenAI SDK
*   **Bundler:** PyInstaller
